ما هو Flutter؟ ولماذا Flutter؟

التعريف

Flutter هو إطار عمل مفتوح المصدر من تطوير Google يُستخدم لبناء تطبيقات متعددة المنصات (Android, iOS, Web, Desktop) من قاعدة كود واحدة باستخدام لغة البرمجة Dart.

لماذا نختار Flutter؟

  • أداء عالي: Flutter لا يستخدم WebView بل يرسم واجهات مستخدم أصلية (Native)
  • تطوير سريع: ميزة Hot Reload تمكنك من رؤية التغييرات فورًا
  • واجهة جميلة: مجموعة غنية من Widgets الجاهزة والقابلة للتخصيص
  • مجتمع نشط: دعم قوي من Google ومجتمع المطورين
  • منصة واحدة: كود واحد يعمل على جميع المنصات

مثال واقعي

تطبيقات شهيرة مثل Google Ads و Alibaba و BMW تم بناؤها باستخدام Flutter، مما يثبت كفاءته في المشاريع الكبيرة.

Cross-Platform vs Native

المعيار Cross-Platform (Flutter) Native (Java/Kotlin, Swift)
عدد أكواد التطوير كود واحد لجميع المنصات كود منفصل لكل منصة
الأداء أداء قريب من Native أفضل أداء ممكن
وقت التطوير أسرع بكثير أبطأ (يجب كتابة الكود مرتين)
التكلفة أقل تكلفة أعلى تكلفة
المرونة واجهة موحدة لجميع المنصات تحكم كامل بكل منصة

نصيحة للمطورين

اختر Flutter عندما تحتاج لتطوير تطبيق يعمل على منصات متعددة بوقت وتكلفة أقل، واختر Native عندما تحتاج لأقصى أداء أو ميزات خاصة بمنصة معينة.

ما هي Dart ولماذا تُستخدم مع Flutter؟

مقدمة عن Dart

Dart هي لغة برمجة كائنية التوجه (OOP) تم تطويرها بواسطة Google، تتميز ب:

  • سهولة التعلم (تشبه Java وJavaScript)
  • أداء عالي (تدعم JIT وAOT Compilation)
  • Null Safety (تقلل من الأخطاء الشائعة)
  • دعم ممتاز للبرمجة غير المتزامنة (async/await)

تثبيت Dart SDK باستخدام Chocolatey

الخطوات الكاملة

  1. افتح PowerShell كمسؤول (Run as Administrator)
  2. قم بتشغيل الأمر التالي لتثبيت Chocolatey:
    Set-ExecutionPolicy Bypass -Scope Process -Force;
    [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072;
    iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
  3. بعد الانتهاء، اختبر التثبيت بـ:
    choco -v
  4. قم بتثبيت Dart SDK:
    choco install dart-sdk
  5. تحقق من التثبيت:
    dart --version

المتغيرات في Dart و Null Safety

أنواع المتغيرات

// التصريح عن المتغيرات
var name = 'أحمد'; // نوعه String
int age = 25; // عدد صحيح
double height = 1.75; // عدد عشري
bool isStudent = true; // true/false
final city = 'القاهرة'; // لا يمكن تغييرها
const PI = 3.14; // ثابت في وقت الترجمة

Null Safety

في Dart، المتغيرات لا يمكن أن تكون null إلا إذا صرحت بذلك صراحة باستخدام علامة ?

// بدون Null Safety
int x; // خطأ، يجب أن تكون initialized

// مع Null Safety
int? y; // صحيح، يمكن أن تكون null
y = null; // مسموح
y = 10; // مسموح

// التحقق من null
if (y != null) {
  print(y + 5); // آمن
}

لماذا Null Safety؟

Null Safety تقلل من الأخطاء الشائعة في البرمجة التي تنتج عن محاولة الوصول إلى متغيرات null، مما يجعل التطبيقات أكثر استقرارًا.

العوامل في Dart

العوامل الحسابية

int a = 10;
int b = 3;

print(a + b); // 13 (الجمع)
print(a - b); // 7 (الطرح)
print(a * b); // 30 (الضرب)
print(a / b); // 3.333... (القسمة العادية)
print(a ~/ b); // 3 (القسمة الصحيحة)
print(a % b); // 1 (الباقي)

العوامل المنطقية

bool x = true;
bool y = false;

print(x && y); // false (و)
print(x || y); // true (أو)
print(!x); // false (نفي)

العوامل الشرطية

int age = 20;
String status = age >= 18 ? 'بالغ' : 'قاصر';
print(status); // 'بالغ'

تمرين تطبيقي

اكتب برنامج يحسب مساحة الدائرة باستخدام العوامل الحسابية (المساحة = π × نصف القطر²)

عوامل الزيادة والنقصان

int x = 5;

// Post-increment
print(x++); // 5 (يتم الطباعة ثم الزيادة)
print(x); // 6

// Pre-increment
print(++x); // 7 (يتم الزيادة ثم الطباعة)

// Post-decrement
print(x--); // 7
print(x); // 6

// Pre-decrement
print(--x); // 5

عوامل التخصيص

int a = 10;
a += 5; // a = a + 5 → 15
a -= 3; // a = a - 3 → 12
a *= 2; // a = a * 2 → 24
a ~/= 4; // a = a ~/ 4 → 6 (قسمة صحيحة)
a %= 4; // a = a % 4 → 2

جمل التحكم (Control Flow)

if - else

int score = 85;

if (score >= 90) {
  print('ممتاز');
} else if (score >= 80) {
  print('جيد جداً');
} else if (score >= 70) {
  print('جيد');
} else {
  print('يحتاج تحسين');
}

switch - case

String grade = 'B';

switch (grade) {
  case 'A':
    print('ممتاز');
    break;
  case 'B':
    print('جيد جداً');
    break;
  default:
    print('درجة غير معروفة');
}

Loops (الحلقات)

// for loop
for (int i = 1; i <= 5; i++) {
  print('الرقم: $i');
}

// while loop
int count = 0;
while (count < 3) {
  print('مرحباً');
  count++;
}

// do-while loop
int x = 5;
do {
  print(x);
  x--;
} while (x > 0);

📜 String Methods في Dart

أهم دوال النصوص (Strings)

1. length - طول النص

String name = 'Flutter';
print(name.length); // 7

2. toUpperCase() / toLowerCase() - تحويل الحروف

String text = 'Dart Programming';
print(text.toUpperCase()); // 'DART PROGRAMMING'
print(text.toLowerCase()); // 'dart programming'

3. contains() - التحقق من وجود نص

String message = 'Welcome to Flutter';
print(message.contains('Flutter')); // true
print(message.contains('Dart')); // false

4. substring() - استخراج جزء من النص

String lang = 'Dart Programming';
print(lang.substring(5)); // 'Programming'
print(lang.substring(0, 4)); // 'Dart'

5. split() - تقسيم النص إلى قائمة

String fruits = 'Apple,Banana,Orange';
List<String> fruitsList = fruits.split(',');
print(fruitsList); // ['Apple', 'Banana', 'Orange']

دوال متقدمة للنصوص

6. trim() - إزالة المسافات الزائدة

String spaced = ' Flutter ';
print(spaced.trim()); // 'Flutter'
print(spaced.trimLeft()); // 'Flutter '
print(spaced.trimRight()); // ' Flutter'

7. replaceAll() - استبدال النصوص

String text = 'I like Java';
String newText = text.replaceAll('Java', 'Dart');
print(newText); // 'I like Dart'

8. startsWith() / endsWith() - التحقق من البداية والنهاية

String url = 'https://flutter.dev';
print(url.startsWith('https')); // true
print(url.endsWith('.com')); // false

9. indexOf() / lastIndexOf() - البحث عن النصوص

String sentence = 'Dart is awesome and Dart is fun';
print(sentence.indexOf('Dart')); // 0
print(sentence.lastIndexOf('Dart')); // 17
print(sentence.indexOf('Python')); // -1 (غير موجود)

10. padLeft() / padRight() - إضافة حشوة للنص

String num = '5';
print(num.padLeft(3, '0')); // '005'
print(num.padRight(3, '*')); // '5**'
