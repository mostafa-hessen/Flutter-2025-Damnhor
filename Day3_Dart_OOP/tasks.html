Dart Programming Tasks - OOP Concepts

Task 1: Class and Object

Objective: Create a class and instantiate objects.
  1. Create a class called Car with the following properties:
    • brand (String)
    • model (String)
    • year (int)
  2. Create an object of the Car class and set values for the properties.
  3. Print the values of the properties of the object.
Additional Challenge:
Take object information once using a constructor and another time using ordinary assignment.

Task 2: Named and Optional Constructors

Objective: Practice using named and optional constructors in Dart.
  1. Create a class called Book with the following properties:
    • title (String)
    • author (String)
    • price (double)
  2. Create the following constructors:
    • A default constructor that initializes the properties with named parameters
    • A named constructor called discountedBook that applies a 20% discount to the price
  3. Instantiate objects using both constructors and print their details.

Task 3: Encapsulation

Objective: Practice encapsulation in Dart.
  1. Create a class called Student with the following properties:
    • name (String)
    • age (int)
    • grade (String)
  2. Encapsulate the age property (make it private and provide getters/setters)
  3. Create an object of Student and demonstrate encapsulation

Task 4: Bank Account with Encapsulation

Objective: Implement encapsulation with getters and setters.
  1. Create a class called BankAccount with:
    • pass (String)
    • name (String)
    • adminName (String, private)
    • adminPass (String, private)
    • users (List, private)
  2. Encapsulate:
    • adminName, adminPass, and users
    • Use a setter to add a user
    • Use a getter to retrieve users
  3. Implement two ways:
    • Using functions (e.g., addUser(), getUsers())
    • Using setters and getters (e.g., set user, get users)

Task 5: Inheritance & Polymorphism (Animals Example)

Objective: Apply OOP concepts with different animal behaviors.
  1. Create classes for three animals:
    • Dog
    • Fish
    • Eagle
  2. Common properties:
    • name (String)
    • age (int)
  3. Different behaviors:
    • walk() (Dog)
    • swim() (Fish)
    • fly() (Eagle)
  4. Implement inheritance (e.g., a base Animal class)
  5. Use polymorphism to call different behaviors

Task 6: Method Overriding

Objective: Practice inheritance and method overriding.

    • Inherits all properties from Dad
    • Overrides the work() method ‚Üí prints "Works as a Software Engineer"
    • Modifies the name property (e.g., adds a nickname)
