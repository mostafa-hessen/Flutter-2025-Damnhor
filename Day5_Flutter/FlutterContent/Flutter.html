ما هو الـ Widget في Flutter؟

في Flutter، كل شيء عبارة عن Widget! هذه هي الفلسفة الأساسية لبناء الواجهات في Flutter. الـ Widget هو لبنة البناء الأساسية التي تصف كيف يجب أن تبدو واجهة التطبيق.

category أنواع الـWidgets

flash_on

Stateless Widget

لا يتغير مع الوقت (لا يحتوي على حالة)

autorenew

Stateful Widget

يتغير بناءً على تفاعل المستخدم أو الزمن (له حالة)

// مثال على Stateless Widget
class MyText extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Text('Hello World!');
  }
}

// مثال على Stateful Widget
class Counter extends StatefulWidget {
  @override
  _CounterState createState() => _CounterState();
}

class _CounterState extends State<Counter> {
  int count = 0;
  
  @override
  Widget build(BuildContext context) {
    return ElevatedButton(
      onPressed: () {
        setState(() {
          count++;
        });
      },
      child: Text('Count: $count'),
    );
  }
}

الـ Widgets في Flutter

في Flutter، كل شيء تقريبًا هو Widget! وهي الوحدات الأساسية لبناء واجهة المستخدم (UI).

يمكن تقسيم الـ Widgets إلى عدة مجموعات لتبسيط فهمها:

Layout Widgets (لترتيب العناصر)

الـ Widget الاستخدام
Column ترتيب العناصر عموديًا (من أعلى لأسفل)
Row ترتيب العناصر أفقيًا (من اليسار لليمين)

Stateful vs Stateless Widgets

toggle_off StatelessWidget

  • Widget لا يتغير (ثابت)
  • UI لا يتغير خلال الاستخدام
  • أسرع في الأداء

toggle_on StatefulWidget

  • Widget يمكن أن يتغير (ديناميكي)
  • UI يتغير حسب البيانات أو التفاعل
  • يستخدم في النماذج والبيانات المتحركة

الـ Widgets الأساسية

الـ Widget الوصف مثال
Text عرض نص على الشاشة Text('Hello', style: TextStyle(fontSize: 20))
Image عرض صورة من الإنترنت أو الجهاز Image.network('https://example.com/image.jpg')
Icon عرض أيقونة من مكتبة Material Icons Icon(Icons.star, color: Colors.yellow)
ElevatedButton زر قابل للنقر مع تأثير الارتفاع ElevatedButton(onPressed: () {}, child: Text('Click'))
TextField حقل إدخال نص TextField(decoration: InputDecoration(labelText: 'Name'))
ListView قائمة قابلة للتمرير ListView(children: [Item1, Item2, Item3])

مثال تطبيقي كامل

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: MyHomePage(),
    );
  }
}

class MyHomePage extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('تطبيق Flutter'),
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: <Widget>[
            Text(
              'مرحباً بكم في Flutter!',
              style: TextStyle(
                fontSize: 24,
                fontWeight: FontWeight.bold,
              ),
            ),
            SizedBox(height: 20),
            Icon(
              Icons.flutter_dash,
              size: 50,
              color: Colors.blue,
            ),
            SizedBox(height: 20),
            ElevatedButton(
              onPressed: () {
                print('تم النقر على الزر!');
              },
              child: Text('انقر هنا'),
            ),
          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () {},
        tooltip: 'إضافة',
        child: Icon(Icons.add),
      ),
    );
  }
}
