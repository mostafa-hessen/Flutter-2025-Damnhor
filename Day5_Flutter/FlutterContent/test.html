1. ThemeData
palette

ThemeData في Flutter

الـ ThemeData هو كائن يحدد المظهر العام لتطبيق Flutter، حيث يحتوي على جميع الإعدادات المتعلقة بالألوان، الخطوط، الأشكال، وغيرها من الخصائص المرئية.

info الخصائص الرئيسية

  • colorScheme: مجموعة الألوان الأساسية للتطبيق
  • textTheme: أنماط النصوص المستخدمة في التطبيق
  • appBarTheme: إعدادات شريط التطبيق
  • buttonTheme: إعدادات الأزرار
  • inputDecorationTheme: إعدادات حقول الإدخال

lightbulb كيفية الاستخدام

يتم تمرير ThemeData إلى widget MaterialApp ليطبق على كامل التطبيق:

MaterialApp(
  theme: ThemeData(
    // إعدادات المظهر هنا
  ),
) 
// مثال شامل لـ ThemeData
ThemeData(
colorScheme: ColorScheme.light(
primary: Colors.blue,
secondary: Colors.green,
),
textTheme: TextTheme(
headline1: TextStyle(fontSize: 32, fontWeight: FontWeight.bold),
bodyText1: TextStyle(fontSize: 16, color: Colors.black87),
),
appBarTheme: AppBarTheme(
color: Colors.blue,
titleTextStyle: TextStyle(fontSize: 20, fontWeight: FontWeight.bold),
),
elevatedButtonTheme: ElevatedButtonThemeData(
style: ElevatedButton.styleFrom(
primary: Colors.blue,
padding: EdgeInsets.symmetric(horizontal: 20, vertical: 12),
),
),
inputDecorationTheme: InputDecorationTheme(
border: OutlineInputBorder(),
filled: true,
fillColor: Colors.grey[100],
),
)

فوائد استخدام ThemeData

  • تطبيق مظهر موحد على كامل التطبيق
  • سهولة تغيير المظهر العام من مكان واحد
  • توفير الوقت في تكرار إعدادات المظهر
  • دعم أفضل لوضع الظلام (Dark Mode)
2. Buttons in Flutter (جميع الأنواع)

أزرار Flutter - جميع الأنواع

يوفر Flutter عدة أنواع من الأزرار لكل منها استخدام محدد ومظهر مختلف. فيما يلي الأنواع الرئيسية للأزرار في Flutter:

arrow_upward

ElevatedButton

زر مرتفع مع ظل، يستخدم للإجراءات الرئيسية في التطبيق.

ElevatedButton(onPressed: () {}, child: Text('زر'))
text_fields

TextButton

زر نصي بدون ارتفاع، يستخدم للإجراءات الثانوية.

TextButton(onPressed: () {}, child: Text('زر'))
crop_square

OutlinedButton

زر بإطار، يجمع بين ميزات ElevatedButton و TextButton.

OutlinedButton(onPressed: () {}, child: Text('زر'))

compare مقارنة بين أنواع الأزرار

النوع الاستخدام المظهر
ElevatedButton الإجراءات الرئيسية والأكثر أهمية خلفية ملونة مع ظل
TextButton الإجراءات الثانوية والأقل أهمية نص فقط بدون خلفية
OutlinedButton إجراءات متوسطة الأهمية إطار مع نص بدون خلفية
IconButton أزرار الأيقونات في AppBar أيقونة فقط
FloatingActionButton الإجراء الأساسي في الشاشة زر دائري عائم
3. Expanded
expand

Expanded في Flutter

الـ Expanded هو widget يستخدم داخل Column أو Row لملء المساحة المتاحة، حيث يأخذ الطفل المساحة المتبقية بناءً على عامل التمدد (flex factor).

info الخصائص الرئيسية

  • child: الـ Widget الذي سيتم توسيعه
  • flex: عامل التمدد (يحدد نسبة المساحة التي سيأخذها)

lightbulb حالات الاستخدام الشائعة

  • عندما تريد أن يأخذ عنصر مساحة أكبر من العناصر الأخرى
  • لتوزيع المساحة بين عدة عناصر بنسب مختلفة
  • لضمان استغلال كامل المساحة المتاحة
// أمثلة على استخدام Expanded
// 1. توزيع المساحة بالتساوي
Row(
children: [
Expanded(child: Container(color: Colors.red, height: 100)),
Expanded(child: Container(color: Colors.green, height: 100)),
Expanded(child: Container(color: Colors.blue, height: 100)),
],
)

// 2. توزيع المساحة بنسب مختلفة
Column(
children: [
Expanded(
flex: 2, // يأخذ ضعف المساحة
child: Container(color: Colors.amber),
),
Expanded(
flex: 1, // يأخذ نصف المساحة
child: Container(color: Colors.purple),
),
],
)

الفرق بين Expanded و Flexible

Expanded هو نوع من Flexible مع ضبط fit ليكون FlexFit.tight (يأخذ كل المساحة المتاحة)، بينما Flexible يمكن ضبطه ليكون FlexFit.loose (لا يأخذ كل المساحة بالضرورة).

4. ListTile
list

ListTile في Flutter

الـ ListTile هو Widget متخصص لعرض صفوف في القوائم، يوفر تخطيطًا قياسيًا للعناصر مع أيقونة وعنوان ووصف فرعي.

info الخصائص الرئيسية

  • leading: عنصر في البداية (عادة أيقونة)
  • title: النص الرئيسي للعنصر
  • subtitle: النص الثانوي (اختياري)
  • trailing: عنصر في النهاية (عادة أيقونة)
  • onTap: دالة عند النقر على العنصر

star مميزات ListTile

  • تخطيط متوافق مع إرشادات Material Design
  • دعم جيد لإمكانية الوصول (Accessibility)
  • تفاعل افتراضي عند النقر (ripple effect)
  • سهولة في إنشاء قوائم متسقة
// أمثلة على استخدام ListTile
// 1. عنصر قائمة أساسي
ListTile(
leading: Icon(Icons.email),
title: Text('البريد الإلكتروني'),
subtitle: Text('user@example.com'),
trailing: Icon(Icons.arrow_forward),
onTap: () {
// تنفيذ إجراء عند النقر
},
)

// 2. قائمة إعدادات
Column(
children: [
ListTile(
leading: Icon(Icons.settings),
title: Text('الإعدادات'),
),
ListTile(
leading: Icon(Icons.notifications),
title: Text('الإشعارات'),
trailing: Switch(value: true, onChanged: (v) {}),
),
ListTile(
leading: Icon(Icons.help),
title: Text('المساعدة'),
),
],
)
